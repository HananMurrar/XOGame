### XO game 
Its simple react project that implements XO game 

#### Steps:
##### 1. Create the next.js app: 
      npx create-next-app@latest xo-game

##### 2. Move into project folder: 
      cd xo-game

##### 3. Run development server: 
      npm run dev

##### 4. Open your browser and go to: 
      http://localhost:3000
